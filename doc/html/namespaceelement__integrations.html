Performs the integration over the element:

\[ \boldsymbol{f}_b = \int_{\Omega^{el}}\boldsymbol{N}^{T}S\text{d}\Omega \]

Parameters
x_nodesAn \(N_{en}\times 2\) numpy array of node positions for the element where \( N_{en} \) is the number of elemental nodes.
elAn element module providing the functions el.N() and el.gauss_quad().
SA function that takes in a size 2 numpy array representing a position [ \( x,y\) ] and returns a scalar value.
ngpThe number of Gauss Points to be used for the quadrature.
Returns
\( \int_{\Omega^{el}}\boldsymbol{N}^{T}S\text{d}\Omega \), a numpy array of size \(N_{en}\)

Performs the integration over the element:

\[ \boldsymbol{f}_\Gamma = \int_{\Gamma}\boldsymbol{N}^{T}\boldsymbol{n}\cdot\boldsymbol{q}\text{d}\Gamma \]

Parameters
x_nodesA \(2 \times 2\) numpy array of node positions at either end of the edge.
nA size 2 numpy array of unit length representing the outward facing normal of the edge
qA function returning a size 2 numpy array representing the flux on the edge. The function takes in a size 2 numpy array representing position in terms of [ \(x,y\)]
ngpThe number of Gauss Points to be used for the quadrature.
Returns
\( \int_{\Gamma}\boldsymbol{N}^{T}\boldsymbol{n}\cdot\boldsymbol{q}\text{d}\Gamma \), a numpy array of size \( 2 \)
