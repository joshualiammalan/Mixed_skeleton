A 2D scalar FEM code
Example

In order to illustrate how this 2D scalar FEM library should be used an example problem is given here. Consider the diagram below

The diagram illustrates a temperature conduction problem with the governing equation

\[ \nabla (-\boldsymbol{D}\nabla T) - S = 0\,, \]

where \(T\) is the temperature, \(\boldsymbol{D}\) is the conductivity matrix and \(S\) is a source term. In this problem the conductivity matrix and the source term are given by

\[ \boldsymbol{D}=\boldsymbol{I}\,,\qquad S=-5(x^2 + y^2)\,, \]

respectively. Additionally, the following boundary conditions hold

\[ \overline{T}_1 = 0 \]

\[ \overline{T}_2 = 5y^2 \]

\[ \overline{q}_1 = 2\begin{bmatrix}x & y\end{bmatrix}^T \]

\[ \overline{q}_2 = \begin{bmatrix}10 & 0\end{bmatrix}^T \]
