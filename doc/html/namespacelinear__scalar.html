Detailed Description

A module containing functions for a scalar linear triangle.

def linear_scalar.B (   x_nodes,
  xi 
)

Defines the gradient of the shape functions in terms of \( x \) and \( y \):

\[ \boldsymbol{B} = \left[ \begin{matrix} \frac{\partial \boldsymbol{N}}{\partial x} \\ \frac{\partial \boldsymbol{N}}{\partial y} \end{matrix} \right] \]

Hint: You'll need J and GN.

Parameters
x_nodesA 3x2 numpy of node positions for the element:
Local
Node num 		x y
1 x1 y1
2 x2 y2
3 x3 y3

Returns
A numpy array of size 2x3 containing the value of the shape function derivatives.
